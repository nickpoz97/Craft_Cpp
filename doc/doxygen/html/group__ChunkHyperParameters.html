Detailed Description

Note
a value of 1 represents 9 chunks (1 front, 1 back, 1 left, 1 right, 1 where tha Camera is located and 4 on the angles)
value of 0 represents 1 Chunk only (the Chunk where the Camera is located)
in practise each of these values indicates a square of Chunks centered on Camera::getPq() where each side is 2*n+1
Warning
each value should be positive
