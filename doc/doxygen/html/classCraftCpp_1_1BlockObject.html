Detailed Description

template<unsigned nFaces>
class CraftCpp::BlockObject< nFaces >

Class that builds Chunk 's blocks.

Template Parameters
nFacesnumber of faces of this block
Note
every block should have 6 faces, but plant does not have the upper and lower one
Warning
this class is used to fill an external array of block vertices and it only contains iterators to that vertices (no arrays in BlockObject objects) for optimization
